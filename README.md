# Learn SASS

Este repositório foi criado para estudar e praticar SASS/SCSS, um pré-processador CSS que ajuda a escrever estilos de forma mais organizada, reutilizável e poderosa.

## Objetivo
- Aprender conceitos básicos e avançados de SASS.
- Explorar variáveis, mixins, funções e aninhamento.
- Criar exemplos práticos de estilização.

## Estrutura do Projeto
- `scss/` → Arquivos fonte em SASS/SCSS.
- `css/` → Arquivos compilados para CSS.
- `index.html` → Página de exemplo para testar os estilos.

## Tecnologias
- [SASS](https://sass-lang.com/)
- HTML5
- CSS3
